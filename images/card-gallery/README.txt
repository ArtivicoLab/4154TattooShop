CARD GALLERY FOLDER
==================

This folder is for the interactive card deck gallery.

HOW TO ADD NEW CARDS:
1. Simply drop new tattoo images into this folder
2. The website will automatically detect them and create new cards
3. Supported formats: .jpg, .jpeg, .png, .webp

IMAGE NAMING CONVENTION:
- Use descriptive names like: "colorful-dragon-tattoo.jpg"
- The filename will be used to generate the card title
- Avoid special characters and use hyphens instead of spaces

EXAMPLES:
- black-gray-portrait.jpg → "Black Gray Portrait"
- vibrant-flower-sleeve.jpg → "Vibrant Flower Sleeve"
- custom-geometric-design.png → "Custom Geometric Design"

The cards will appear automatically when the page loads!
